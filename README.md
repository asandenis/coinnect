<h2 id="installation" style="color: #4A90E2;">⚙️ Installation</h2>
<p>Follow these steps to install the project:</p>

<pre>
<code>
git clone https://github.com/asandenis/coinnect.git
cd coinnect/server
npm install express, mongoose, cors, bcrypt, multer, jsonwebtoken, path, node-cron
cd ../client
npm install react-scripts
</code>
</pre>

<h2 id="usage" style="color: #4A90E2;">🚀 Usage</h2>
<p>To start using the project:</p>
<p>Add valid credentials to server/server.js (MongoDB database).</p>
<p>Then run:</p>

<p>First terminal (for the server):</p>

<pre>
<code>
cd server
node server.js
</code>
</pre>

<p>Second terminal (for the client):</p>

<pre>
<code>
cd client
npm start
</code>
</pre>
